# Звукові файли

## ✅ Поточний стан:

Звукові файли додані та активні!

## Наявні файли:

1. ✅ **click.wav** - Звук кліку по клікеру
2. ✅ **purchase.wav** - Звук покупки
3. ✅ **background.wav** - Фонова музика (loop)

## Формат:

- Використовується формат: **WAV**
- Всі звуки оптимізовані для мобільних пристроїв

## Налаштування:

Звуки можна налаштувати в меню "Налаштування":
- Увімкнення/вимкнення звукових ефектів
- Увімкнення/вимкнення фонової музики
- Регулювання гучності звуків (0-100%)
- Регулювання гучності музики (0-100%)
