# Звукові файли

Додайте наступні звукові файли в цю папку:

## Необхідні файли:

1. **click.mp3** - Звук кліку по клікеру (короткий, ~0.1-0.2 сек)
2. **purchase.mp3** - Звук покупки (позитивний звук, ~0.3-0.5 сек)
3. **background.mp3** - Фонова музика (loop, ~1-3 хв)

## Рекомендації:

- Формат: MP3 або WAV
- Розмір: якомога менший (оптимізуйте для мобільних пристроїв)
- Звуки повинні бути не занадто гучними

## Де знайти безкоштовні звуки:

- [Freesound.org](https://freesound.org/)
- [Zapsplat.com](https://www.zapsplat.com/)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)

## Поточний стан:

⚠️ Файли поки що відсутні. Додаток працюватиме без звуків, поки ви їх не додасте.
